CSS ShortCuts
Explanation

A Stylesheet with a name for properties from flex to cursor to font.

There are some elements where users only want to put one or two properties, this helps make the property be matched to the element by class name instead of having space taken in the css file or space. It has the properties from the groups: flex, justify, align, float, display, position, list, border, text, font, color, background, and cursor. And more will be added!

This stylesheet was made for me to be able to have fast CSS properties but I decided to share it with the world. Anyone can use it as long as they say it's theirs
Code Sample 1

This sample is most of the Shortcut file but not all. It is meant to make you see that there are lots of properties as for right and in the future.

They are all class names so that they can be used multiple times in the same page and they are also easily accessible.

Every class name are meant to be as small as possible while still keeping the name describing the property.

All this file is made only in CSS and works perfectly and better than intended. All you have to do is write the name into a class attribute and voila. The property is now added to the element.
